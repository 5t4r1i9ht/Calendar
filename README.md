# Calendar

## 程序设计课程项目：桌面日历

使用Qt 5.9开发。

项目要求：

- [x] 能按周、月的方式显示日历
- [x] 能显示农历和节假日
- [x] 能作为记事本使用，记录每一天的日常行为
